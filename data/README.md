# Dataset notes

This directory contains the input files used to reproduce the analyses in this repository. Two public datasets are included after formatting for the CPGP workflow.

## Lee2019 cattle dataset

The cattle data were derived from the study by Lee et al. (2019):

Lee, J., Kim, J. M., & Garrick, D. J. (2019). Increasing the accuracy of genomic prediction in pure-bred Limousin beef cattle by including cross-bred Limousin data and accounting for an F94L variant in MSTN. *Animal Genetics*, 50(6), 621-633. https://doi.org/10.1111/age.12846

Original data source: https://doi.org/10.6084/m9.figshare.8298674.v1

The files used by this repository are stored under `data/Lee2019/`.

## Xie2021 pig dataset

The pig data were derived from the study by Xie et al. (2021):

Xie, L., Qin, J., Rao, L., Tang, X., Cui, D., Chen, L., Xu, W., Xiao, S., Zhang, Z., & Huang, L. (2021). Accurate prediction and genome-wide association analysis of digital intramuscular fat content in longissimus muscle of pigs. *Animal Genetics*, 52(5), 633-644. https://doi.org/10.1111/age.13121

Original data source: https://doi.org/10.6084/m9.figshare.14274785.v1

The files used by this repository are stored under `data/Xie2021/`.

## Use of the data

The data files in this repository are provided only to reproduce the analyses reported in the corresponding CPGP study. For any other research use, users should follow the terms, licences and data-use conditions specified by the original data providers. When required, users should obtain permission directly from the data providers or contact the corresponding authors of the original publications.
